# Asmar's Photography — asmarphotography.com

Static website for Muhammad Asmar Hussain — landscape photographer, Sony Brand Ambassador, TEDx speaker, Co-Founder of Travelbug.

- `index.html` — main site (hero, TEDx talk, "As Seen On", portfolio coverflow, print shop, FAQ, contact)
- `wall-preview.html` — "See It On Your Wall" print room preview
- `gallery/`, `assets/` — photographs and site assets
- `robots.txt`, `sitemap.xml`, `llms.txt` — SEO / AI-search
- `vercel.json` / `_redirects` — old-URL 301 redirects + caching

Pure static — no build step. Deploy by serving the folder as-is.
